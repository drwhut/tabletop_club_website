Pre-release Builds

This page includes downloads for the latest pre-release version of Tabletop Club for each platform. Pre-release builds are releases specifically designed to help test new features, and to get feedback from the community about certain tweaks.

Instructions on how to download and play the pre-release builds on each platform can be found in the documentation.

Just looking for the latest stable release? Then you will want to visit the downloads page.
